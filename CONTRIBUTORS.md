## Ancestor Projects

- **Get Shit Done (GSD)** — The predecessor project from which PAN Wizard was derived. GSD established the core architecture: CLI dispatcher, phase-based workflow, agent system, state management, and the command/agent/core layered design. PAN Wizard evolved GSD's `gsd-tools.cjs` into `pan-tools.cjs`, expanded from 11 to 24 core modules (through v3.4.0), added 5-runtime support, and introduced the focus-design pipeline, standards system, context engineering features, Opus 4.7 integration (extended thinking, prompt caching, cross-phase agent memory in v2.10), and Spec B v2 (cost dashboard, foresight, deep review, knowledge commands, counterfactual exploration, MCP bridge, and hierarchical orchestration in v3.0-v3.4).
