<task type="auto">
  <name>Task 2: Create file walker module</name>
  <files>src/walker.js</files>
  <action>
Create src/walker.js that exports a `walk(dir)` function:

1. Use `const fs = require('node:fs')` and `const path = require('node:path')`
2. `walk(dir)` accepts a directory path string, returns `string[]` of absolute .md file paths
3. Implementation:
   - Use `fs.readdirSync(dir, { withFileTypes: true })` to get directory entries
   - For each entry:
     - If `entry.isDirectory()`: recurse into `path.join(dir, entry.name)`
     - If `entry.isFile()` and `path.extname(entry.name) === '.md'`: add `path.join(dir, entry.name)` to results
     - Otherwise: skip (this naturally skips symlinks since we only check isFile/isDirectory)
   - Sort the final array alphabetically with `.sort()` for deterministic output
4. Export: `module.exports = { walk }`
5. Handle the case where `dir` does not exist: let the fs error propagate (CLI handles it)

Do NOT use `{ recursive: true }` option — use manual recursion as specified in context.md.
Do NOT follow symlinks — `dirent.isFile()` and `dirent.isDirectory()` return false for symlinks.
  </action>
  <verify>
node -e "const { walk } = require('./src/walker'); const fs = require('node:fs'); const path = require('node:path'); const tmp = path.join(__dirname, 'fixtures', 'sample'); fs.mkdirSync(path.join(tmp, 'subdir'), { recursive: true }); fs.writeFileSync(path.join(tmp, 'a.md'), '# A'); fs.writeFileSync(path.join(tmp, 'subdir', 'b.md'), '# B'); fs.writeFileSync(path.join(tmp, 'skip.txt'), 'not md'); const files = walk(tmp); assert(files.length === 2); assert(files[0].endsWith('a.md')); assert(files[1].endsWith(path.join('subdir', 'b.md'))); console.log('Walker OK')"
  </verify>
  <done>src/walker.js exports walk() that recursively finds .md files, excludes non-.md, sorts alphabetically, does not follow symlinks</done>
</task>

<task type="auto">
  <name>Task 3: Create engine stub and formatters</name>
  <files>src/engine.js, src/formatters/human.js, src/formatters/json.js</files>
  <action>
**src/engine.js:**
Create engine module that exports a `lint(filePaths, readFile)` function:
1. `filePaths`: string[] of absolute file paths
2. `readFile`: function(filePath) => string (dependency injection, not used in Phase 1 stub)
3. Returns `LintResult[]` where each result is `{ filePath: string, violations: [] }`
4. In Phase 1, the engine has no rules. It maps each filePath to `{ filePath, violations: [] }`
5. Export: `module.exports = { lint }`

The `readFile` parameter establishes the contract for Phase 2 when rules need file content. In Phase 1, the engine does not call it.

**src/formatters/human.js:**
Create human formatter that exports a `format(results)` function:
1. `results`: LintResult[] (array of `{ filePath, violations[] }`)
2. For each violation: emit `${filePath}:${v.line}:${v.column}: ${v.severity} ${v.rule} ${v.message}`
3. When no violations exist across all results: return empty string
4. When violations exist: join lines with `\n` and add trailing `\n`
5. Use `path.relative(process.cwd(), filePath)` for readable file paths in output
6. Export: `module.exports = { format }`

**src/formatters/json.js:**
Create JSON formatter that exports a `format(results)` function:
1. `results`: LintResult[] (same shape)
2. Return `JSON.stringify(results) + '\n'`
3. With zero violations, results is `[{ filePath, violations: [] }, ...]` — stringify produces valid JSON
4. Export: `module.exports = { format }`
  </action>
  <verify>
node -e "const { lint } = require('./src/engine'); const r = lint(['/tmp/a.md', '/tmp/b.md'], () => ''); assert(r.length === 2); assert(r[0].violations.length === 0); const hf = require('./src/formatters/human'); assert(hf.format(r) === ''); const jf = require('./src/formatters/json'); const j = JSON.parse(jf.format(r)); assert(Array.isArray(j)); assert(j.length === 2); console.log('Engine + Formatters OK')"
  </verify>
  <done>engine.js returns LintResult[] with empty violations; human formatter returns empty string for zero violations; json formatter returns valid JSON array</done>
