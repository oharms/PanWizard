**bin/panmd.js:**
Create the CLI entry point with shebang `#!/usr/bin/env node`:

1. `'use strict';` at top
2. Require modules:
   - `const { parseArgs } = require('node:util')`
   - `const path = require('node:path')`
   - `const fs = require('node:fs')`
   - `const { walk } = require('../src/walker')`
   - `const { lint } = require('../src/engine')`

3. Wrap main logic in try/catch. On any caught error: write error message to stderr, set `process.exitCode = 2`.

4. Parse args with `util.parseArgs`:
   ```javascript
   const { values, positionals } = parseArgs({
     args: process.argv.slice(2),
     options: {
       format: { type: 'string', default: 'human' },
       rules: { type: 'string' },
       fix: { type: 'boolean', default: false },
       help: { type: 'boolean', default: false },
       version: { type: 'boolean', default: false },
     },
     allowPositionals: true,
     strict: true,
   });
   ```

5. Handle `--help`: write help text to stderr (NOT stdout), return. Help text should show:
   ```
   Usage: panmd lint <dir> [options]

   Options:
     --format <human|json>  Output format (default: human)
     --rules <rule,...>     Run only specified rules
     --fix                  Auto-fix violations where possible
     --help                 Show this help
     --version              Show version
   ```

6. Handle `--version`: read version from package.json, write to stderr, return.

7. Validate command: `positionals[0]` must be `'lint'`. If not, write "Unknown command" to stderr, set exitCode = 2, return.

8. Validate target directory: `positionals[1]` must be provided. Resolve with `path.resolve(positionals[1])`. If directory doesn't exist (check with `fs.existsSync` + `fs.statSync().isDirectory()`), write error to stderr, set exitCode = 2, return.

9. Walk: `const filePaths = walk(targetDir)`

10. Lint: `const results = lint(filePaths, (fp) => fs.readFileSync(fp, 'utf8'))`

11. Format: Load the formatter based on `values.format`:
    - `'human'` → `require('../src/formatters/human')`
    - `'json'` → `require('../src/formatters/json')`
    - Anything else → write "Unknown format" to stderr, exitCode = 2, return
    - Call `formatter.format(results)` to get output string

12. Output: `process.stdout.write(output)` — only if output is non-empty

13. Exit code: Check if any violation has `severity === 'error'`. If yes: `process.exitCode = 1`. If no violations: `process.exitCode = 0` (default).

ALL diagnostic messages (help, version, errors, warnings) go to `process.stderr.write()`.
ONLY violation output goes to `process.stdout.write()`.
