<objective>
Build the parse + validate pipeline that turns a flow file path into a verified Flow IR. Loader handles file I/O and shape normalization. Validator handles schema validation (aggregated errors) and cycle detection (iterative Kahn with leftover-DFS path recovery).

Purpose: This is the gate that PARSE-01..PARSE-05 enforce. By the end of this plan, every malformed flow is rejected before any spawn happens, and every well-formed flow has been normalized so downstream code (Plan 04 scheduler) never has to handle `undefined` `depends_on` or string-coerced `attempts`.
Output: `loadFlow(path)` and `validateFlow(flow)` functions, an exhaustive fixture catalog, and a test suite that covers both happy paths and the M5 + C1 pitfall families.

Create `src/loader.js` exporting `loadFlow(filePath)`. This function:

1. **Read the file** with `readFileSync(filePath, 'utf8')`. If `ENOENT`, throw `new ValidationError(\`flow file not found: ${filePath}\`)` (NOT a `RuntimeError` — file-not-found is a user input problem, exit 2 per CLI-10).
2. **Parse JSON** with `JSON.parse`. On `SyntaxError`, throw `new ValidationError(\`malformed JSON in ${filePath}: ${err.message}\`)`. Mitigates **PARSE-01**.
3. **Validate top-level shape:**
   - If parsed value is `null` or not an object → `throw new ValidationError(\`flow file must be a JSON object with a "tasks" array\`)`.
   - If `flow.tasks` is `undefined` or `null` → same error message.
   - If `flow.tasks` is not an array → `throw new ValidationError(\`flow.tasks must be an array (got ${typeof flow.tasks})\`)`.
   - If `flow.tasks` is an empty array → return a valid Flow IR with `{tasks: {}, ...}` (empty flow is a no-op success per pitfalls.md M5; validator will produce empty topoOrder).
4. **Normalize each task** into the canonical Flow IR shape. For each task in `flow.tasks`:
   - If task is not an object → throw `new ValidationError(\`tasks[${i}]: must be an object\`)`.
   - If `task.id` is missing OR not a string → throw `new ValidationError(\`tasks[${i}].id: must be a non-empty string\`)`.
   - **Normalize `depends_on`:** if missing/null/undefined, set to `[]`. If present and not an array, throw `new ValidationError(\`tasks[${i}].depends_on: must be an array (got ${typeof task.depends_on})\`)`. (DO NOT validate ID references here — that's the validator's job. Just shape.)
   - **Normalize `retry`:** if missing, set to `{attempts: 1, backoff_ms: 0}`. If present, leave as-is (validator will check shape).
   - Keep all other fields verbatim (forward-compat for Phase 2's task fields).
5. **Build the IR shape** and return:
   ```js
   return {
     tasks: byId,         // { [id]: normalizedTask }
     children: null,      // filled by validateFlow
     topoOrder: null,     // filled by validateFlow
   };
   ```
   Note: `byId` keys preserve insertion order from the source array — important for SCHED-05 (deterministic final state).

    errors.push({ path: `tasks[${id}].id`, issue: 'must be a non-empty string (no whitespace-only)' });
  }

  if (!/^[A-Za-z0-9_][A-Za-z0-9_./:\-]*$/.test(task.id)) {
    errors.push({ path: `tasks[${id}].id`, issue: `invalid characters (must match /^[A-Za-z0-9_][A-Za-z0-9_.\\/:\\-]*$/)` });
  }
  // cmd: required string (PARSE-02)
  if (typeof task.cmd !== 'string' || task.cmd.length === 0) {
    errors.push({ path: `tasks[${id}].cmd`, issue: 'must be a non-empty string' });
  }
  // depends_on: each entry must be a string (PARSE-03 — id type)
  for (let i = 0; i < task.depends_on.length; i++) {
    const dep = task.depends_on[i];
    if (typeof dep !== 'string') {
      errors.push({ path: `tasks[${id}].depends_on[${i}]`, issue: `must be a string id (got ${typeof dep})` });
      continue;
    }
    if (!flow.tasks[dep]) {
      errors.push({ path: `tasks[${id}].depends_on[${i}]`, issue: `references unknown task "${dep}"` });
    }
  }
  // retry shape (PARSE-04): if present, must be {attempts: int >= 1, backoff_ms: int >= 0}
  if (task.retry !== undefined && task.retry !== null) {
    if (typeof task.retry !== 'object' || Array.isArray(task.retry)) {
      errors.push({ path: `tasks[${id}].retry`, issue: 'must be an object {attempts, backoff_ms}' });
    } else {
      if (!Number.isInteger(task.retry.attempts) || task.retry.attempts < 1) {
        errors.push({ path: `tasks[${id}].retry.attempts`, issue: `must be integer >= 1 (got ${JSON.stringify(task.retry.attempts)})` });
      }
      if (!Number.isFinite(task.retry.backoff_ms) || task.retry.backoff_ms < 0 || !Number.isInteger(task.retry.backoff_ms)) {
        errors.push({ path: `tasks[${id}].retry.backoff_ms`, issue: `must be integer >= 0 (got ${JSON.stringify(task.retry.backoff_ms)})` });
      }
    }
  }
}

if (errors.length > 0) {
  const summary = errors.map(e => `  - ${e.path}: ${e.issue}`).join('\n');
  throw new ValidationError(`flow validation errors:\n${summary}`, errors);

### `detectCycle(flow)` — iterative Kahn with leftover-DFS path recovery

```js
export function detectCycle(flow) {
  const ids = Object.keys(flow.tasks);
  const indegree = Object.create(null);
  const children = Object.create(null);
  for (const id of ids) { indegree[id] = 0; children[id] = []; }
  for (const id of ids) {
    for (const dep of flow.tasks[id].depends_on) {
      indegree[id]++;
      children[dep].push(id);
    }
  }
  flow.children = children;

  // Iterative Kahn — NO recursion, so no RangeError on long chains.
  const queue = [];
  // Seed in deterministic order (insertion order of ids).
  for (const id of ids) if (indegree[id] === 0) queue.push(id);
  const remaining = { ...indegree };
  const topo = [];
  while (queue.length > 0) {
    const id = queue.shift();
    topo.push(id);
    for (const c of children[id]) {
      if (--remaining[c] === 0) queue.push(c);
    }
  }

  if (topo.length === ids.length) {
    flow.topoOrder = topo;
    return topo;
  }

  // Cycle exists — recover one concrete cycle path via iterative DFS over leftovers.
  const leftover = ids.filter(id => remaining[id] > 0);
  const path = recoverCyclePath(leftover, children, ids);
  throw new CycleError(`cycle detected: ${path.join(' → ')}`);
}

function recoverCyclePath(leftover, children, allIds) {
  const set = new Set(leftover);
  // Self-loop edge case: pick the leftover that has itself as a child.
  for (const id of leftover) {
    if (children[id].includes(id)) return [id, id];
  }
  // Iterative DFS, return on first back-edge found.
  // Pick start in deterministic order (first leftover id by insertion).
  const start = leftover[0];
  const stack = [[start, [start]]];
  const onStack = new Set([start]);
  while (stack.length > 0) {
    const [cur, path] = stack[stack.length - 1];
    let advanced = false;
    for (const nxt of children[cur]) {
      if (!set.has(nxt)) continue;
      if (onStack.has(nxt)) {
        // Back-edge — cycle found. Trim path to start at nxt.
        const idx = path.indexOf(nxt);
        return [...path.slice(idx), nxt];
      }
      onStack.add(nxt);
      stack.push([nxt, [...path, nxt]]);
      advanced = true;
      break;
    }
    if (!advanced) {
      stack.pop();
      onStack.delete(cur);
    }
  }
  // Fallback (shouldn't reach here if leftover is non-empty): return start twice.
  return [start, start];
}
```
